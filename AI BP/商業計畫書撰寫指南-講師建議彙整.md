## 一、核心原則：說好故事比堆砌內容更重要

### 黃金圈法則（Golden Circle）— StarFab 劉晏蓉
- 先專注表達「**WHY（為什麼）**」，而非平鋪直述「WHAT（做了什麼）」
- 人已厭煩 AI 創造的文字，**真誠故事的能力反而更珍貴**
- **Mathify 實踐**：以「為什麼台灣 108 萬國高中生在數學學習上的痛苦需要被解決」開篇

### 避免技術導向過重 — 張提提（中華開發資本）
- 技術背景出身的創業家常過份強調技術而忽略市場競爭與銷售計畫
- **不要花一半以上篇幅描述技術功能，而只用不到一頁說明營運模式或市場競爭分析**
- AI/LLM/TikZ/Desmos 是手段而非目的，著重在解決了什麼問題、創造了什麼價值

## 二、營運計畫書架構（七大章節）— 張提提

| 章節 | 內容重點 |
|------|----------|
| 1. 經營計畫概要 | 公司與經營團隊介紹、本次增資額度與資金運用、分階段目標、BP 重點摘要 |
| 2. 公司簡介 | 成立時間、股東結構、技術沿革、主要產品 |
| 3. 團隊/組織簡介 | 學經歷、成功經驗、組織架構、人力資源計劃（含員工認股） |
| 4. 產品/服務說明 | 功能特性、應用領域、發展階段、Roadmap、競爭分析比較、技術優勢與利基 |
| 5. 營運模式 | 市場/客戶、成本與定價、銷售模式（含海內外）、階段目標（Milestone）、策略夥伴 |
| 6. 產業/市場與競爭分析 | 目標市場、市場規模、客戶特徵、競爭者比較表 |
| 7. 財務計劃 | 過去經營的財務報表、未來 3-5 年財務規劃、資本預算、融資計畫、企業評價 |

### 計畫書三大功能
1. **縮短評估時間**：提供投資者所需資訊，讓他們能有效率地篩選分析
2. **展現知己知彼**：公司競爭優劣勢、外部市場環境的機會威脅、策略與風險評估
3. **提供詳細投資報酬分析**：投資者最關心投資報酬、投資風險以及如何回收投資資金

## 三、投資人評估重點

### 創投評估六大構面 — 張提提
1. 公司簡介（基本資料、股本形成、董監結構、經營團隊、募資計畫）
2. 營運概況（產品、營運模式、銷售分析、技術競爭優勢、成長動能）
3. 財務概況（損益表、資產負債表、現金流量表、售價與成本分析）
4. 產業概況（產業結構/特性、市場規模、發展趨勢）
5. 財務預測（預估損益表/資產負債表/現金流量表、同業比較）
6. 風險分析（SWOT、產業/技術/團隊風險）

### 投資人的核心問題 — Arthur Chen（BE Health Ventures）
- 第一個問題：**你的防禦性（Barrier to Entry）在哪裡？**
- 盡職調查（DD）要確認 BP 及 PPT 中的資訊是否真確，挖掘產業/業務/技術/財務/法律/人事/股權/IPR 問題
- 也觀察創業者的**個性、價值觀、領導力**

### 創投有興趣的公司特性 — 張提提
- 創新性高：具有獨特技術或商業模式
- 可擴展性：能夠快速擴大業務規模
- 高成長潛力：有潛力成為市場領先者
- 強大的團隊和管理能力
- 清晰的商業模式和市場需求
- **基本報酬率要求 IRR 20%，意即公司估值 4 年內須翻倍**

### 投資人關心重點 — AppWorks 程九如
1. **策略一致性與定位**：核心能力、綜效潛力、長期目標、資源排擠
2. **市場與競爭分析**：市場規模與增長率、潛在客戶分析、競爭者分析、進入與退出障礙、UVP
3. **財務可行性與量化評估**：NPV、IRR、投資回收期、損益兩平點、敏感性分析
4. **營運與執行能力**：核心團隊、組織架構、關鍵資源、供應鏈風險、法規遵循
5. **風險與退出策略**：風險識別與量化、備援計畫、監控指標、退出條件與處置方式

## 四、募資策略與財務規劃

### 募資關鍵數字 — Johnny Yu（Mosaic Venture Lab）
| 項目 | 建議 |
|------|------|
| 募多少 | **24 months runway**（足夠燒 24 個月的資金） |
| 募多久 | **6 months roadshow** |
| 股權稀釋 | **15-25%** 是合理的 |
| 投資人期望報酬率 | **5x-10x** |

### 估值方法 — Johnny Yu
- 估值基礎：財務報表、營收假設、商業模式（價值主張）、市場規模（TAM/SAM/SOM）、市場痛點（PMF）
- 估值方式：市銷率（P/S）、股價淨值比（P/B）、本益比（P/E）、**現金流量折現法（DCF）**
- **里程碑不同，估值就不同**：商業模式決定里程碑，里程碑影響估值

### 里程碑設定七步驟 — Johnny Yu
1. 確立公司階段性目標及最終目標
2. 列出所有專案及細節（who, what, when, where, which）
3. 釐清里程碑的定義（各部門一起決定、風險評估）
4. 創造合適的時間表（如果作業時間不可控，不應設定此里程碑）
5. 釐清資源分配並調整
6. 確認每個階段可達成的指標（KPI 需量化，串聯至各部門及個人）
7. 當達到時記得慶祝

### Term Sheet 關鍵條款 — Arthur Chen
- **經濟條款**：估值與投資金額、證券類型、清算優先權、反稀釋、ESOP 池、優先認購權、拖帶/共售權
- **控制條款**：先決條件、競業禁止、創辦人鎖定/歸屬、董事會、保留事項、資訊權

### 台灣募資現狀 — 張提提
- 困境：極低的新創出場率（約 0.4%）、次級市場優異表現造成資金排擠、創投資金集中追逐半導體
- **A 輪是最困難的「三不管」地帶**
- 資金來源路徑：Family/Friends/Fools（種子/天使輪）→ 新創加速器/天使投資人 → 財務型創投 → 企業創投

### 募資實戰心得 — 張提提
- 募資是場**長期抗戰**，需有充足的糧食及中長期的策略規劃
- **減少簡報內容的錯誤**，細心處理每份文件
- **定期與募資對象更新近況**，當下無法投資不代表永遠無法投資
- 需有**明確出場計畫**，設想 7-10 年後可實現的目標

## 五、商業模式設計

### 商業模式畫布九宮格 — Johnny Yu
- 九大區塊：關鍵合作夥伴、關鍵活動、價值主張、顧客關係、目標客層、關鍵資源、通路、成本結構、收益流
- **價值主張**定義：滿足客戶哪些需求、引起什麼興趣、創造哪些價值、解決哪些問題
- **收益流範圍**：資產銷售、會員費、使用費/租賃/授權、仲介/廣告

### AI 新創的靈魂拷問 — StarFab 劉晏蓉
- **生存**：你們解決的痛點是「剛需」還是「可有可無」？你們的 AI 帶來的價值是否能「自動升級」？
- **擴張**：你們的 BD 流程是否能快速複製？你們是「高效率獲利」還是「高營收低毛利」？
- **差異化**：拓展國際市場的「第一個切入點」在哪裡？你們是客戶的「戰友」還是「委外專案承包商」？

### 第一性原理的商業策略 — StarFab 劉晏蓉
- **WHY**：我們的存在，解決了什麼不可替代的本質問題？
- **HOW**：收費模式是否能與客戶的價值體驗對等？將營收拆解為「**流量 x 轉換率 x ARPU**」精準鎖定並優化弱項
- **WHAT**：持續交付「優化的成果」而非一次性的「AI 工具」。將 PoC 轉化為可複製的長期合約
- 募資階段，高**淨營收留存率（NRR）**是證明產品具備長期生命力與可規模化 BD 效率的鐵證

### 產品定位 — Johnny Yu
- 定位語句架構：**針對「誰」的「什麼問題」，我們提供「什麼產品」來創造「什麼價值」**
- 三種定位策略：價格導向（Cost Leadership）、差異化（Differentiation）、利基市場（Niche Market）
- **Mathify 適合利基市場策略**：專注 K-12 數學，以深度垂直整合和專業可信度取勝

## 六、市場分析方法

### TAM/SAM/SOM 框架 — Johnny Yu, SparkLabs
- TAM：總市場需求（理論最大值）
- SAM：你的產品可服務的區塊
- SOM：短期內你能拿到的市場份額（3-5 年）
- **自下而上（Bottom-Up）通常比較精準**：「單價 x 客戶數 x 使用頻率」
- **Mathify 應用**：108 萬學生 x 60% 補習率 x 目標滲透率 x 月費 NT$600 = SOM

### 三種市場分析工具 — Johnny Yu
1. **SWOT 分析**：找出產品與市場的互動關係
2. **市場區塊（Segmentation）**：針對不同族群特徵分類（人口統計/行為/地理/心理變數）
3. **使用者旅程圖（User Journey Map）**：認知→評估→購買→使用→回購/推薦，每階段標出痛點與機會點

### 市場痛點分析 — Johnny Yu
- 痛點分三種：**功能性**（使用不舒服）、**情緒性**（影響自我評價）、**社會性**（感到落伍）
- CB Insight 調查：100 家失敗新創中 **42% 是「沒有人需要這個產品」**
- 需區分「需要被滿足的需求」vs「未被滿足的需求」，並確認是否為「剛性需求」

### 海外市場調研三大支柱 — SparkLabs 簡夢瑤
- 用戶訪談（質化，5-8 位）
- 用戶調查 & NPS（量化，50-100 位）
- 網站 A/B Testing（100-200 次轉換）
- 核心秘訣：**詢問現在行為，而非想像行為**。「你上一次遇到這個問題是什麼時候？當時你怎麼做的？」

## 七、國際市場拓展策略

### 三維市場評估矩陣 — SparkLabs 簡夢瑤
| 維度 | 評估問題 |
|------|----------|
| 市場潛力 | 市場夠大嗎？客戶願意付費嗎？成長動能如何？ |
| 進入難度 | 法規門檻、競爭強度、文化契合度 |
| 資源槓桿 | 人脈網絡、營運成本、政策與資本 |

- 核心洞察：資源槓桿不是關於你有多少錢，而是**能不能透過一個市場的成果加速另一個市場的進入**

### 進軍日本市場三大法則 — SparkLabs
1. 建立「日本視角」：服務如何回饋日本社會
2. 身在現場、展現決心：團隊中必須有日本在地成員
3. **關係大於數據**：由具信譽的第三方引薦，面對面建立溫度

### 東南亞市場要點 — SparkLabs
- 人口基數不等於願付價格，需用「目標客戶數量 x 願付價格」計算真實 TAM
- 每個國家都是獨立市場
- 進入策略：**先深後廣**，在第一個市場完整驗證 PMF 再複製

### Founder-Led Marketing — SparkLabs
- 創辦人透過個人品牌驅動業務增長，人比品牌更容易獲得信任
- **持續性 > 完美性**，每週固定發文比偶爾發布完美內容更重要

## 八、新創管理工具

### 麥肯錫 7S 模型 — 郭承裕（StarFab 首席顧問）

| 要素 | 一句話理解 |
|------|-----------|
| Strategy 策略 | 怎麼贏 |
| Structure 結構 | 誰決定 |
| Systems 系統 | 怎麼做 |
| Shared Values 共享價值 | 為何存在 |
| Style 領導風格 | 老闆怎麼帶 |
| Staff 人員 | 是誰在做 |
| Skills 技能 | 擅長什麼 |

- 最低分的兩項通常是組織瓶頸，優先改善
- **產品或技術可以協助企業成功，但管理在於防止企業失敗**
- 創業靠機會、技術起步，但要靠管理才走得遠

## 十一、Mathify 商業計畫書十大行動建議

1. **故事先行**：用黃金圈法則（WHY → HOW → WHAT），以學生數學學習痛點開篇
2. **控制技術篇幅**：技術說明不超過全文 20%，重點在價值創造
3. **營收拆解公式**：流量 x 轉換率 x ARPU = 營收，每個變數提供數據支撐
4. **市場分析用 Bottom-Up**：108 萬學生 x 60% 補習率 x 目標滲透率 x NT$600/月 = SOM
5. **製作競爭者比較表**：均一教育、AmazingTalker、其他數學 APP，列出設立時間/營收/產品/技術/估值
6. **財務預測 3-5 年**：損益表、資產負債表、現金流量表，含 DCF 估值
7. **量化里程碑與 KPI**：每個 KPI 串聯至部門和個人，含時間軸
8. **風險分析與退出策略**：SWOT + 風險識別 + 備援計畫
9. **資金運用計畫**：NT$300 萬的具體分配（對應評分標準第五項），設定至少 3 個量化 KPI
10. **國際化願景**：用三維市場評估矩陣評估潛在海外市場（日本、東南亞），展現擴張潛力
