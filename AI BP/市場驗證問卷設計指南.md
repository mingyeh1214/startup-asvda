# 市場驗證問卷設計指南 — Mathify

> 本指南教你如何設計一份有效的市場驗證問卷，附上可直接使用的完整題目。

---

## 一、問卷設計的核心原則

### 1. 問「行為」，不問「意願」

| ❌ 不好的問題 | ✅ 好的問題 |
|-------------|-----------|
| 你會不會用 AI 數學學習工具？ | 你的孩子上次遇到不會的數學題，是怎麼解決的？ |
| 你願意每月花多少錢？ | 你目前每月花多少錢在孩子的數學學習上？ |
| 你覺得這個產品好不好？ | 你上次為了孩子數學問題最困擾的是什麼？ |

**為什麼**：人們會高估自己的購買意願（研究顯示高達 3-5 倍），但描述自己的實際行為時比較誠實。

### 2. The Mom Test 原則

來自 Rob Fitzpatrick 的經典著作《The Mom Test》，三個核心原則：

1. **談他們的生活，不是你的產品**：不要一上來就介紹 Mathify
2. **問具體事實，不是假設**：「上次」比「通常」更有價值
3. **少說多聽**：你的目標是收集資訊，不是推銷

### 3. 漏斗式問題設計

```
第一層：篩選題 → 確認受訪者是目標客群
第二層：行為題 → 了解他們目前怎麼做
第三層：痛點題 → 挖掘不滿和需求
第四層：解方題 → 測試產品概念
第五層：付費題 → 驗證付費意願
第六層：行動題 → 產生實際承諾
```

### 4. 問卷長度

- **線上問卷**：15-20 題以內，5-8 分鐘完成
- **完成率警戒線**：超過 10 分鐘，完成率會掉到 20% 以下
- **黃金比例**：選擇題 70% + 量表題 20% + 開放題 10%

## 二、你需要兩份問卷

Mathify 有兩個關鍵角色：**使用者（學生）** 和 **付費決策者（家長）**。他們關心的事情不同，需要分開調查。

| 對象 | 關心什麼 | 問卷重點 |
|------|---------|---------|
| 家長 | 成績提升、CP 值、安全性 | 消費行為、付費意願、痛點 |
| 學生 | 好不好用、有沒有趣、能不能解決問題 | 學習行為、工具偏好、體驗期望 |

## 五、問卷發放策略

### 目標樣本數

| 對象 | 最低有效樣本 | 建議目標 | 原因 |
|------|------------|---------|------|
| 家長 | 100 份 | 200 份 | 付費決策者，需要足夠的統計顯著性 |
| 學生 | 50 份 | 100 份 | 使用者行為驗證 |

### 發放管道

| 管道 | 操作方式 | 預期效果 |
|------|---------|---------|
| **家長 Line 群組** | 請認識的老師/家長幫忙轉發 | 精準觸及、回收率高 |
| **FB 家長社團** | 親子教育、各縣市家長社團 | 量大、但需要篩選 |
| **PTT / Dcard** | 家庭版、考試版 | 年輕家長為主 |
| **學校合作** | 與老師合作在班級發放 | 學生問卷最佳管道 |
| **Google Forms** | 製作成線上表單 | 免費、方便統計 |

### 提高回收率的技巧

1. **開場白要簡短有吸引力**：「花 3 分鐘填問卷，幫助改善孩子的數學學習」
2. **提供誘因**：填完問卷可抽 NT$100 超商禮券（準備 10-20 份即可）
3. **匿名保證**：強調不會收集個資、資料僅用於研究
4. **即時感謝**：填完後顯示感謝頁面 + 預約體驗的連結

## 六、數據分析重點

問卷回收後，你要從數據中回答以下五個關鍵問題：

### 1. 痛點是否真實存在？
- Q7 中最多人選的困擾是什麼？
- Q9 中「不滿意」的比例有多高？
- **門檻**：超過 50% 受訪者表示不滿意 → 痛點確實存在

### 2. 目前的解決方案是否不足？
- Q4 + Q8：現在用什麼方式？滿意嗎？
- **門檻**：超過 40% 對現有方案不滿意 → 有替換空間

### 3. 產品概念是否被接受？
- Q11：「有幫助」的比例
- Q12：最受歡迎的功能是哪些
- **門檻**：超過 60% 認為有幫助 → 概念被接受

### 4. 付費意願是否存在？
- Q14：願意付費的比例和金額分佈
- Q15：付費觸發條件
- **門檻**：超過 30% 願意付費 NT$400+ → 付費意願存在
- **關鍵計算**：
  ```
  預估付費轉換率 = 願意付費比例 ÷ 3（實際通常是問卷意願的 1/3）
  預估 ARPU = 加權平均願付金額 × 0.7（打 7 折更接近現實）
  ```

### 5. 是否有行動承諾？
- Q16：留聯絡方式的比例
- **門檻**：超過 15% 願意留資料 → 真實需求
- **這是整份問卷最重要的指標**，因為留資料是「行動」不是「意見」

## 七、在商業計畫書中呈現結果

收到數據後，在商業計畫書的「市場驗證」段落中，建議這樣呈現：

```
市場驗證結果：
• 調查對象：X 位國高中生家長
• 70% 的家長表示孩子在家遇到數學問題時無法得到有效協助（痛點驗證）
• 65% 的家長對現有的數學學習解決方案不滿意（替換空間）
• 介紹產品概念後，80% 的家長認為 AI 視覺化解題工具有幫助（概念接受度）
• 45% 的家長表示願意每月支付 NT$400-600（付費驗證）
• X% 的受訪者主動留下聯絡方式，希望參與免費體驗（行動驗證）
```

## 八、進階技巧：超越問卷的驗證方法

問卷只是市場驗證的第一步。更強的驗證方式如下：

| 階段 | 方法 | 證明力 | 成本 |
|------|------|--------|------|
| 1 | ✅ 問卷調查（你現在在這裡） | ★★☆☆☆ | 低 |
| 2 | Landing Page + 留資料 | ★★★☆☆ | 低 |
| 3 | 免費 Beta + 使用數據 | ★★★★☆ | 中 |
| 4 | 付費試用 | ★★★★★ | 中 |

**建議下一步**：問卷回收後，對留下聯絡方式的家長發出 Beta 測試邀請，取得更強的驗證數據。
